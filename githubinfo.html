    All About Github

    Q. What is a Verison control System or VCS?
    A. Tracks the history of changes as people and teams collaborate on projects together.
    Q. What is Git?
    A. Git is a free and open source distributed version control system to handle everything from small to very large projects with speed and efficiency.
    Q. What is Github?
    A. Github is a code hosting platform for version control and collaboration.
    Q. What is the difference between Git and Github?
    A. Git is a verison control system that allows developers to track changes in their code. Github is a web-based hosting service for git respository.
    Q. Who started Github and how was it started?
    A. The founders of Github were Tom Preston-Werner, Chris Wanstrath, P.J Hyett, and Scott Chacon.
    Q. What company owns it now?
    A. Microsoft currently owns Github.
    Q. How much does a Github account cost?
    A.Github accounts are free for basic individuals. For team or advanced collaborations and organization is $3.67/ per month for the first 12 months. For enterprises is $19.25/ per month for the first 12 months
    Q. What is the Octocat?
    A. Octocat is the mascot of the source code hosting service Github.
    Git/Github terms:
    • Respository: Contains all of your project's files and each file's revision history.
    • Commit: Records changes to one or more files in your branch.
    • Fork: A new respository that shares code and visibility setting with the orignal "upstream" respository.
    • Push: Updates the remote branch with loocal comits.
    • Pull Requests: Let you tell others about changes you've pushed to a branch in a respository on Github.
    • Workflows: Is a congfigurable automated process that will run one or more jobs.
    • Issues: Let you track your own work on Github, where development happens.
    • Raw Button: You can view or copy the raw content of a file without any styling.
    • Blame Button: Used to examine the contents of a file line by line and see when each line was last modified and who the author of the modifications was.
